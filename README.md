# Chat Farmacia - Plantilla

Pasos rápidos:

1. Crear proyecto en Supabase y obtener la cadena de conexión.
2. Ejecutar en Supabase:
   - `db/schema.sql`
   - `db/seed.sql`
3. Copiar `.env.example` a `.env.local` y rellenar:
   - `DATABASE_URL`
   - `OPENAI_API_KEY`
   - `APP_SECRET`
4. Instalar dependencias:

```bash
npm install
```

5. Ejecutar en desarrollo:

```bash
npm run dev
```

6. Abrir `http://localhost:3000` y entrar con:

- Alumno demo: `alumno001@demo.local` / `Alumno123!`
- Profesor demo: `profesor@demo.local` / `Profe123!`
